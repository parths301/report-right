# ReportRight

**Expert RBI & FEMA Filing Services**

ReportRight is a modern, responsive landing page for a consultancy firm specializing in Foreign Exchange Management Act (FEMA) and Reserve Bank of India (RBI) compliance. We simplify the complex landscape of foreign investment reporting for Indian entities.

![ReportRight Preview](https://via.placeholder.com/800x400?text=ReportRight+Preview)

## Features

-   **Comprehensive Service Showcase**: Detailed information on key filings:
    -   **Form FC-GPR**: For Foreign Direct Investment (FDI) via capital instruments.
    -   **Form FC-TRS**: For transfer of shares between residents and non-residents.
    -   **LLP Forms**: Form LLP-I and LLP-II for Limited Liability Partnerships.
    -   **Form DI**: For Downstream Investment declarations.
-   **Interactive Pricing**: Clear, tiered pricing plans (Basic, Standard, Enterprise) with hover effects.
-   **Dynamic UI**:
    -   **Dark Mode**: Fully supported system-aware dark mode with a manual toggle.
    -   **Animations**: Smooth reveal-on-scroll animations and typewriter effects.
    -   **Glassmorphism**: Modern frosted glass aesthetic for cards and navigation.
-   **Contact Form**: Functional inquiry form with validation and submission simulation.

## Tech Stack

-   **HTML5**: Semantic structure.
-   **Tailwind CSS**: Utility-first styling (loaded via CDN for easy deployment).
-   **Vanilla JavaScript**: Lightweight logic for animations, theme toggling, and form handling.
-   **FontAwesome**: Icons for visual enhancement.
-   **Google Fonts**: Inter and Playfair Display typography.

## Setup & Usage

### Local Development
1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/report-right.git
    ```
2.  Navigate to the project folder:
    ```bash
    cd report-right
    ```
3.  Open `index.html` in your preferred browser.
    -   *Tip*: Use a live server extension (like Live Server in VS Code) for the best experience.

### Deployment (GitHub Pages)
1.  Push your code to a GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Under **Build and deployment**, select **Source** as `Deploy from a branch`.
4.  Select `main` (or `master`) branch and `/root` folder.
5.  Click **Save**. Your site will be live in a few minutes!
